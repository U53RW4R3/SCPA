# Gobuster

## Package Manager

Install it according to your package manager.

```
# apt install -y gobuster
```

## Universal Install

```
$ go install github.com/OJ/gobuster/v3@latest && \
sudo mv ~/go/bin/gobuster /usr/local/bin
```
