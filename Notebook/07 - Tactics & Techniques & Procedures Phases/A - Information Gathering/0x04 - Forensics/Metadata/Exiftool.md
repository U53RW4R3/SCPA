# Exiftool

> [!TIP]
> Sometimes metadata may contain credentials.


Extract metadata.

```
$ exiftool file.extension
```

Allow duplicate tags to be extracted.

```
$ exiftool -a -u file.extension
```

Extract unknown tags.

```
$ exiftool -u file.extension
```
