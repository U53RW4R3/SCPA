# 09 - Comments

Single-line comment.

```sql
SELECT -- This is a comment
FROM table_name;

SELECT -- This is select command
FROM Sales
WHERE year = 2020;
```

Multi-line comment.

```sql
/* This is a multi-line comment */
SELECT * FROM table_name;

/* This comment is long and spans multiple lines. */
SELECT *
FROM Sales
WHERE year = 2015;
```
