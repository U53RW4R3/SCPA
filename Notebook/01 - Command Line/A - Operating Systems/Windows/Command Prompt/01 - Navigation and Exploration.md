# 01 - Navigation and Exploration

## 1.1 - List Files and Directories

### 1.1.1 - List files

```
C:\> dir
```

Find files with a specific extension

```
C:\> dir /s *.txt
```

## 1.2 - Change Directories

### 1.2.1 - Change directory

```
C:\> cd folder

C:\> chdir folder
```
