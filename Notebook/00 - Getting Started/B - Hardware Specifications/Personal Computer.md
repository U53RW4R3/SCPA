# Personal Computer

## Desktop

### Minimum Specs

| Hardware Component | Requirement                  |
| ------------------ | ---------------------------- |
| CPU                | AMD Ryzen 3 or Intel Core i3 |
| RAM                | 8GB                          |
| Storage            | 1TB HDD                      |
| GPU                | Integrated CPU               |

### Recommended Specs

| Hardware Component | Requirement                               |
| ------------------ | ----------------------------------------- |
| CPU                | AMD Ryzen 5 (quad-core) or Intel Core i5. |
| RAM                | 32GB                                      |
| Storage            | 1TB SSD                                   |
| GPU                | NVIDIA RTX or AMD Radeon RX series.       |
