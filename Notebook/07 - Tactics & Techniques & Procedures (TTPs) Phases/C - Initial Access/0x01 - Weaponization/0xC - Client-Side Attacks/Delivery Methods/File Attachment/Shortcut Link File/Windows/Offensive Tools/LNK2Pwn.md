# LNK2Pwn

## 01 - Setup

```
$ wget https://github.com/it-gorillaz/lnk2pwn/releases/download/v1.0.0/lnk2pwn-1.0.0.zip && \
unzip lnk2pwn-1.0.0.zip
```

## 02 - Usage

```
$ java -jar lnk2pwn-1.0.0.jar &
```
