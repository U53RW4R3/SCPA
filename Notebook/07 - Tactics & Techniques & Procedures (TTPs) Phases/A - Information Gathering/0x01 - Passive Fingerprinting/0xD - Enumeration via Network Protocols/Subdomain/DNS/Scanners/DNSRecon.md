# DNSRecon

## Usage

```
$ dnsrecon -d <domain.com> -a -n <DNS_IP>
```

```
$ dnsrecon -d <domain.com> -t axfr
```

```
$ dnsrecon -r <IP_DNS>/<CIDR> -n <IP_DNS>
```
