# 05 - Armory

Search Tag(s): #sliver #command-and-control

## 5.1 - Help Menu

```
sliver > armory -h
```

## 5.2 - Usage

List all the armory extensions.

```
sliver > armory
```

Install an armory externsion.

```
sliver > armory install <extension>
```

Install all armory extensions

```
sliver > armory install all
```
