# Urless

## 01 - Setup

```
$ pip install git+https://github.com/xnl-h4ck3r/urless.git -v
```

## 02 - Usage

`$ urless -i urls.txt -o truncated-output.txt`

## 03 - Use Cases

### Truncate Endpoints

`$ waybackurls <domain.com> | urless -o fuzz-endpoints-output.txt`
