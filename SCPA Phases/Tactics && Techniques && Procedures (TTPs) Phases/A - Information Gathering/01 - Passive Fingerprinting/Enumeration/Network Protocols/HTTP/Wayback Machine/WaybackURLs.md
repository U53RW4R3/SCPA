# WaybackURLs

## 01 - Setup

```
$ go install github.com/tomnomnom/waybackurls@latest && \
sudo mv ~/go/bin/waybackurls /usr/local/bin
```

## 02 - Usage

`$ waybackurls <website.com> > urls.txt`

`$ cat domains.txt | waybackurls > urls.txt`
