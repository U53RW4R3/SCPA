# Uro

## Setup

```
$ git clone https://github.com/s0md3v/uro && cd uro && \
sudo python3 setup.py install
```

## Usage

`$ uro -i urls.txt -o truncated-output.txt`

## Use Cases

### Truncate Endpoints

`$ waybackurls <domain.com> | uro -o truncated-fuzz-endpoints-output.txt`
