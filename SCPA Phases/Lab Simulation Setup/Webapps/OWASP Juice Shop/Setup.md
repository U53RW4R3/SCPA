# Setup

Search Tag(s): #lab #juice-shop #webapp-pentesting

```
$ git clone https://github.com/juice-shop/juice-shop.git --depth 1 && \
cd juice-shop
```

## DNS Hosts Addresses

Map the OWASP juice shop IPv4 address as a DNS reference.

```
$ sudo nano /etc/hosts
<web_server_IP> juice.shop
```
